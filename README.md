🎯 Milestone Objective

Milestone 2 focuses on transforming raw climate data into meaningful analytical insights and designing high-impact visualizations that clearly communicate weather trends, seasonal behavior, extreme events, and regional similarities. This milestone bridges data analysis and visual storytelling, forming the analytical backbone of ClimateScope.

🔬 Core Analytical Work
📊 Statistical Analysis

Computed mean, skewness, interquartile range (IQR) and distribution metrics for selected climate variables

Analyzed data distributions and variability to assess climate stability

Implemented rolling averages to capture underlying long-term trends

🌦️ Seasonal Pattern Analysis

Segmented climate data into seasonal groups (Summer, Monsoon, Autumn, Winter)

Compared seasonal means using bar charts and box plots

Identified seasonal fluctuations and consistency across time

⚠️ Extreme Weather Detection

Detected extreme climate deviations using statistical thresholds

Highlighted extreme events on time-series plots

Generated a detailed extreme events table with date, metric value, and weather condition text

🔍 Trend Decomposition (Conceptual)

Applied rolling-window trend extraction

Visualized:

Underlying trend component

Residual (noise / extreme variance)

Enabled intuitive understanding of long-term vs short-term climate behavior

🌍 Comparative Climate Analysis
🔁 Regional Side-by-Side Comparison

Compared multiple countries using:

Statistical range plots

Time-series overlays

Rainfall share visualization

Enabled direct visual comparison of regional climate characteristics

🧠 Climate Similarity Index

Designed a Similarity Index to quantify how closely two locations resemble each other climatically

Calculated similarity score based on normalized metric variance

Visualized seasonal similarity using a heatmap for intuitive interpretation

📈 Visualization Design Strategy

The following visualization types were carefully selected to maximize insight clarity:

Line Charts → Long-term trends & rolling averages

Bar Charts → Seasonal mean comparisons

Box Plots → Distribution spread & outlier detection

Scatter / Marker Plots → Extreme event highlighting

Heatmaps → Seasonal and regional similarity analysis

KPI Cards → Instant statistical summaries (Health Score, Mean, Skewness, IQR)

Each visualization aligns directly with a specific analytical goal, ensuring interpretability and decision relevance.
